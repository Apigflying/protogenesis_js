classList 修改，增加，或者删除元素的className box.classList 获取到的是一个元素的class集合，是个类数组 ["box", value: "box"] ["add", value: "add"] ["lll", "add2", value: "lll add2"] box.classList.add(className); // 给某个元素添加某个className box.classList.remove(className); // 给某个元素删除某个className box.classList.contains(className); // 判断元素是否有某个className box.classList.toggle(className) // 如果元素有这个className就删除，否则就添加
