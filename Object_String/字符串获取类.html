字符串方法: charAt()方法:传入的是参数类型是数字,查找字符串中对应下标的字符 只能查找字符串中的一位 str.indexOf('f') → 1 相对应的 str.charAt(1) → 'f' (一 ) str.indexOf() 1.传一个参数的时候,参数的类型是字符串 (1).若传入的是字符串,会正常查找 (2).若传入的只有一个数字,那么会将这个数字的类型转换为字符串,然后进行查找 获取 **指定字符串（‘45’）** @首位@在该字符串内（str）内的位置的下标数字，只返回第一次遇到的这个字符串（‘45’）的下标，如果找不到指定字符串，那么返回的值就是-1 2.传入两个参数的时候，第一个是字符串类型，第二个参数是数字类型 console.log(str.indexOf(2,6)) 传入的第一个参数如果是数字,会被默认转换为字符串 传入的第二个参数,是从str这个字符串的**第几位(包括当前写的数字的这一位,即若当前写的这一位就是第一个参数的字符串,那么返回的就是第二个参数本身)**开始找,默认(不写第二个参数的情况下)从该字符串的第0位开始查找 注:如果传入的第二个参数是负数,默认从0开始 indexOf()方法返回的值的数据类型是'number' console.log(str.length) console.log(str.indexOf('',0)) 用indexOf搜索时,若第一个值为''即空字符串 分为两种情况:1)当 第二个值<=str.length时，返回值就是第二个参数 2)当 第二个值>str.length时,返回值就是str.length (二) str.lastIndexOf() 使用方法同indexOf(),区别是,例如: var str='hellow world' str.indexOf('o') → 返回值是:4 str.lastIndexOf('o') → 返回值是:8 lastIndexOf只有一位字符串时,即只有一个参数,默认从后往前找,找到的第一个'o'后,返回当前这个'o'在字符串中的位置(下标) lastIndexOf有两个参数的时候,第二个参数的意思是:从这个数字在字符串中的位置开始,依次从后往前找对应的字符 例如: 上面的str中 var str='hellow world' ↑↑↑↑↑↑↑↑↑↑ 0123456789 str.lastIndexOf(9) → 从第9位字符串开始,依次向前查找(包括第9位字符串本身),当第一次找到'o'时,返回这个'o'在字符串中的下标 → 8 str.lastIndexOf(6) → 4 console.log(str.lastIndexOf('df',0)); 当第二个参数为0时,就是从第0位向前找,那么返回的要么是0(第一位就是要找的)要么是-1 (三) str.includes() //console.log(str.includes(22)) str.includes() 传入的参数是字符串,返回值是true和false 当传入的是数字时,也会被自动转换为字符串类型 (四)str.startsWith()和str.endsWith() 这两个方法传入的参数是字符串,判断str这个字符串是否以**传入的字符串**开头和结尾 返回值是true 和false
